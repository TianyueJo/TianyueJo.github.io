# Tianyue Zhou — Personal Website

Academic homepage for [Tianyue Zhou](https://tianyuejo.github.io/).

## Local preview

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000

## Rebuild CV

```bash
cd cv
pdflatex tianyue.tex
```

## Deploy

This site is published via GitHub Pages from the `main` branch.
